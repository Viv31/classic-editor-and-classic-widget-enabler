== Short Description ==
Enables the Classic Editor and Classic Widgets in WordPress with a single plugin.

== Description ==
The **Classic Editor And Classic Widgets Enabler** plugin allows users to enable both the Classic Editor and Classic Widgets within a single plugin. This eliminates the need to install separate plugins for these features.

### Features:
- Enables the Classic Editor.
- Enables Classic Widgets.
- No additional settings required.
- Simple and lightweight solution.

== Installation ==
1. Download the plugin zip file.
2. Upload it to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. The Classic Editor and Classic Widgets will be enabled automatically.

== Frequently Asked Questions ==

= How does this plugin work? =
This plugin automatically enables the Classic Editor and Classic Widgets upon activation. No additional configuration is needed.

= Why should I use this plugin? =
If you prefer the Classic Editor and Classic Widgets experience, this plugin saves you from installing two separate plugins. It provides a simple, all-in-one solution.

= Does this plugin require any configuration? =
No, the plugin works automatically upon activation.
